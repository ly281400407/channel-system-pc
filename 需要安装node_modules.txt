安装模块：
"electron-updater": "^2.11.0",
"sqlite3": "^3.1.13"

特别说明：
sqlite3需要重新编译
步骤如下
--采用局部安装的方法，不建议采用全局，会出现导入model失败情况
windows-build-tools执行很慢需要耐心等待，具体安装参考网址安装方法。
https://github.com/electron/electron/blob/master/docs-translations/zh-CN/tutorial/using-native-node-modules.md
https://github.com/nodejs/node-gyp
命令行如下：
npm install -g windows-build-tools
npm install -g node-gyp
npm install -g node-pre-gyp
cd 项目目录
npm install --save sqlite3
cd sqlite3所在目录
node-pre-gyp rebuild --target=1.7.8 --target_arch=x64 -target_platform=win32 --dist-url=https://atom.io/download/electron --runtime=electron
node-pre-gyp rebuild --target=1.7.8 -target_platform=win32 --target_arch=ia32 --dist-url=https://atom.io/download/electron --runtime=electron
如果采用全局安装sqlite需要额外配置一个NODE_PATH变量：
NODE_PATH=C:\Users\0003\AppData\Roaming\npm\node_modules
